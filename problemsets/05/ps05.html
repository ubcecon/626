Problem 1

Suppose \(X_n = O_p(a_n)\) and \(Y_n - c = O_p(a_n)\) for \(c \neq 0\) and \(a_n \to 0\). Show that \(\frac{X_n}{Y_n} = O_p(a_n)\).

Hint: \(\frac{X_n}{Y_n} = \frac{X_n}{c} + \frac{X_n}{Y_n} (c - Y_n) \frac{1}{c}\), use Lemma 3.3 and Exericse 3.1 from Song (2021).

Problem 2

Consider a linear model where the dimension of \(X\) might increase with \(n\) \[ Y_i = X_{i(n)} \beta_{(n)} + u_i \] for \(i = 1, ..., n\). Assume that observations are i.i.d. with \(\Er[u u'] = \sigma^2 I_n\), and \(\Er[X_{i(n)} u] = 0\).

To simplify the analysis, further assume that \(X_{i(n)}\) are one of the standard basis vectors, i.e. \(X_{i(n)} \in \{ e_j \}_{j=1}^{k(n)} \subset \R^{k(n)}\) where \(e_1 = (1, 0, ..., 0)\), \(e_2 = (0, 1, 0, ..., 0)\), etc, and that \[ P(X_{i(n)} = e_j ) = \frac{1}{k(n)} \] for each \(j \in \{1, ..., k(n)\}\)

  1. Let \(\hat{\beta} = (X'X)^{-1} (X'y)\) be the OLS estimator. Show that \(\hat{\beta}_j = \frac{1}{\sum_i 1\{X_{i(n)} = e_j\}} \sum_{i=1}^n 1\{X_{i(n)} = e_j\} Y_i\), and \(\hat{\beta}_j - \beta_{j, (n)} = \frac{\frac{1}{n} \sum_i (Y_i - \beta_j) 1\{X_{i(n)} = e_j\}}{\frac{1}{n} \sum_i 1\{X_{i(n)} = e_j\}}\)

  2. Show that \(\norm{ \hat{\beta} - \beta_{(n)} } = O_p\left( \sqrt{k(n)/n} \right)\).

Hint: use the previous part and problem 1.

Problem 3

A hypothesis test has asymptotic size \(\alpha\) if \[ \lim_{n \to \infty} \sup_{P_x \in \mathcal{P}_0} P_x(C_n) = \alpha \] where \(C_n\) is the critical region. Suppose that under \(H_0\) (i.e. for all \(P_x \in \mathcal{P}_0\)), a test statistic, \(T_n(X)\) converges in distribution to \(Z\), uniformly in \(\mathcal{P}_0\).1 Show that if \(P(Z > t) = \alpha\), then letting \(C_n = \{X: T_n(X) > t\}\) leads to a test of asymptotic size \(\alpha\).

References

Song, Kyunchul. 2021. “Introduction to Econometrics.”

Footnotes
